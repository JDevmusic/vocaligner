# VocAligner

## Vision

VocAligner is an AI-powered web application that helps Logic Pro users recreate the vocal production style of their favourite artists using only Logic Pro stock plugins.

The goal is to make professional vocal production accessible to creators without requiring expensive third-party plugins or years of mixing experience.

---

## Mission

Help music creators achieve professional vocal results faster by translating the sound of commercial records into practical Logic Pro plugin chains.

---

## Target Audience

Primary users:

- Logic Pro producers
- Independent artists
- Bedroom producers
- Songwriters
- Mixing beginners
- Content creators

---

## Core Problem

Many producers know how they want their vocals to sound but don't know:

- which plugins to use
- which order to use them
- how to configure the settings

Finding this information requires watching hours of YouTube tutorials or purchasing expensive presets.

VocAligner simplifies this process.

---

## Core Value Proposition

Type:

Artist

Song

↓

Receive a Logic Pro stock plugin chain with suggested settings.

---

## MVP

The first version includes:

- Artist input
- Song input
- AI-generated Logic plugin chain
- Logic-style settings display

No authentication.

No audio upload.

No payments.
