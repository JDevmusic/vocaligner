# VocAligner

## Mission

Allow producers to recreate the vocal production style of their favourite artists using only Logic Pro stock plugins.

## Long-term Vision

Become the leading AI vocal engineering assistant by combining AI research, production knowledge and personalised vocal analysis.

## Core MVP

- Artist input
- Song input
- AI research
- Logic plugin recommendations
- Visual plugin interfaces
- Cached results

## Future Features

- Vocal upload
- Personalised recommendations
- Other DAWs
- Community ratings
- Saved presets
