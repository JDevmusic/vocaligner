VocAligner Principles

Purpose

VocAligner is an intelligent production assistant.

Never optimise for AI.
Always optimise for helping producers.

---

Decision hierarchy

1. Product quality
2. User understanding
3. Architecture
4. Maintainability
5. AI capability

---

Design

Premium.

Calm.

Typography first.

Whitespace is a feature.

One action per screen.

---

Engineering

AI performs judgement.

Code performs mechanics.

Prefer deterministic code.

Avoid duplication.

---

UX

Reduce uncertainty.

Teach rather than impress.

Every screen should build confidence.

---

Features

Critique.

Design.

Implement.

Never implement first.

---

Future

Every component should be reusable.

Every plugin should become interactive.

Audio analysis should extend existing architecture.
